Welcome to MD's documentation!
===================================

**Mod Demon** (/Mahh'Dee'mon/) is a Discord Bot for large and small servers
that makes multi-purpose an under statement
It pulls data from the `database <https://mod-demon.xyz/>`_
and offers a *complex* or *simple* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
